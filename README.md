# Athar's Math Games

Interactive browser games built around Athar's kindergarten math curriculum.

## Games

### Game 1 — Number Clash (`game1-number-clash.html`)

King Kong vs. Godzilla battle game for comparing numbers. Two numbers appear on cards — tap the bigger one to win the round. Covers the "greater than / less than" comparing numbers skill from the curriculum.

**How to play:**
- Choose an arena (City, Ocean, or Jungle)
- 5 rounds of King Kong vs. Godzilla — tap the bigger number
- After all 5 rounds, face the King Cobra in a bonus round
- The hint bar writes out "X is greater than Y" after each correct answer
- Tap **?** for a hint if you're stuck
